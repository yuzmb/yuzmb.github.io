笔试中经常出现的js数组排序与去重算法

JavaScript
  view

数组排序比较常用的：冒泡排序、快速排序、sort()方法排序；数组去重方法也有很多。

冒泡排序：

从数组中随便拿一个数与后一位比较，如果前者比后者大，那么两者交换位置，从而遍历数组可以得到排序的效果

快速排序：

在数组中间那一个值，然后用这个值跟数组里面的值相比较，大于此值的放在一边，小于的也放在一边，然后用concat()合并，再进行比较，如此反复
